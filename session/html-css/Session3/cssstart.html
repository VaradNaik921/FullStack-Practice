Ways to Include CSS:

  1. Inline CSS(Style direct inside Tag)
  2. Internal CSS(Style in head)
  3. External CSS(LINK)
Types of CSS Selectors
1.ID Selector #Id 2.Class Selector .Class 3.Element
*Above is the order of specificity*
*Note: For Id selector use id=(unique id) and to style use #(unique id) and for class=(unique id) and style .(unique id)
*Difference in ID and Class, Class can share its name with other tag but ID has to be unique
*Can give mulitple class to single tag

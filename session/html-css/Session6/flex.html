PARENT PROPERTIES

    Properties of Parent Flex:
  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

    • Direction in Flex: Row, Column, Row Reverse, Column Reverse
    Property:flex-direction
    Default value:Row

    Wrap in Flex: nowrap, wrap, wrap reverse
    Property:flex-wrap
    Default value:nowrap

    Flow in Flex: flex-direction and flex-wrap can be used singlehandedly using flex-flow
    Property:flex-flow
    Default value:row nowrap

    Justify-Content: used for alignment in main axis
    Property:justify-content
    Default value:flex-start
    space-around: margin applied equally on sides of box, space between:space added between the boxes only, space-evenly: equal space added around everywhere

    Align-Items: used for alignment in cross axis(perpendicular to main axis)
    Property:align-items
    Default value:flex-start
